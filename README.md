\# Bangalore Daily Rates Tracker



A daily tracker for gold and fuel prices in Bangalore, scraped from BankBazaar.



\## Features

\- Daily gold rate scraping (22K and 24K)

\- Daily fuel rate scraping (petrol and diesel)

\- SQLite database with both live and official end-of-day figures

\- Interactive Streamlit dashboard with trend charts and KPIs



\## Tech Stack

\- Python (requests, BeautifulSoup, SQLite)

\- Streamlit + Plotly for the dashboard
